# Enum `Resource`

A database resource
A resource is a location, or a range of locations, from which data can be fetched.

## Variants

### `Table`

Table name

### `RecordId`

Record ID

### `Object`

An object

### `Array`

An array

### `Edge`

Edges

### `Range`

A range of id's on a table.

### `Unspecified`

Unspecified resource
